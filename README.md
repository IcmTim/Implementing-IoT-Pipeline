# Implementing-IoT-Pipeline
This repository contains a full-scale IoT pipeline course curriculum anf self-study tasks.
## Research Topics Covered
- Deep Learning Optimization
- Data Pipeline Scalability
- High-performance and supercomputing
- UI/UX Design Methodology
- Applied deep learning. Deep dive into tensor operations and the numerical foundations of AI.
## Tech Stack
- Frameworks: TensorFlow, Keras
- Environment: Jupyter Notebooks
- Visualization: Matplotlib
## Getting Started
Environment Setup:
- *pip install tensorflow numpy matplotlib*
